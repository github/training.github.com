Git Foundations Course

GitHub Training regularly offers open-enrollment online courses. Our Git Foundations course provides you with the training you need to get bootstrapped on Git and to start using it in your daily workflow.

During this course, we'll explore the foundations of Git and the practical, everyday commands you'll need. We'll conclude with some workflow discussion and tips that will set you up for success with this revolutionary version control system.

Prerequisites:

    Basic knowledge about a version control system, whether that be Subversion, CVS, Perforce, StarTeam, SourceSafe, ClearCase or any similar product.

Objectives:

  • Understand how Git works and how to apply that to day-to-day development.
  • Learn the ten basic commands that will appear in everyday usage of Git
  • Know how to "back out" of mistakes using Git's incredible history and ability to revert almost any change.
  • Leverage the collaboration features of Git for productivity boosts with colleagues.
  • Discover how the offline capabilities of Git work "under the hood" and how to use those to work in more places with less reliance on network connectivity.
